to see this file from root system, middleware should add following piece.
  • First add "Microsoft.AspNet.StaticFiles": "1.0.0-rc1-final" NuGet Package
  • app.UseStaticFiles(); // But with this you have to navigate to index.html
  • app.UseDefaultFiles(); // Place before UseStaticFiles() middleware otherwise will not work
  • or use single method app.UseFileServer(); instead of above both app.
